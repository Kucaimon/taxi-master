## Инфра / Vercel / SPA demo

- **Деплой SPA:** переписывание маршрутов приложения на `index`, wildcard rewrite, ранний fallback для прямых URL (`40bd080`, `0577f93`, `a579932`).
- **Демо:** нет вечного спиннера конфига и нет 404 при навигации в SPA (`6d6612f`).
- **Зависимости:** удалён некорректный пакет `-` из `package.json` (`b1963c2`).
- **Старт проекта:** базовая заготовка под Vercel (`685c4f2`), сетап после merge (`1c35560`).

## Fullscreen / хром карты

- **Плавный вход в полноэкранный режим** и корректное скрытие `.layout__header` (`c5587a4`, `e52c8f2`).
- **Нижняя форма остаётся доступной:** в нативном и псевдо-fullscreen, с восстановлением drag при полноэкранной карте (`53ef533`, `501e716`, `e742abc`).
- **Оверлеи пассажира скрываются** для «чистой» карты в true fullscreen (`10c5934`).
- **Кнопки полноэкранного режима и «моё место»** видны в pseudo-fullscreen (`2ea1ef9`).
- **Мобильные браузеры / iOS:** усилен fallback активации fullscreen, надёжнее полноэкранный режим (`062109a`, `150317d`).

## Пассажир: bottom sheet / свайп / раскладка

- **Свайп панели:** Pointer Events вместо устаревших жестов; границы snap для свёрнутого состояния; защита `ResizeObserver` при `offsetParent === null` (`024d764`, `787feb5`, `6453497`).
- **UI во время drag:** не дублировать кнопки (`47e2567`).
- **Высота sheet в px** для стабильности на мобильных (`d600fa8`).
- **Карта на весь bleed по умолчанию;** в fullscreen панель «поднимается»; слот карты по видимой области для корректного центрирования геолокации (`e08cb37`, `5077540`).
- **Пропорции карты / нижней зоны:** 75/25, 85/15, отступы между картой и чипами границ; чипы режимов не висят над картой; безопасные отступы (`d2812e0`, `0754f9f`, `a94ef50`, `baafd41`).
- **Отделение нижней панели** от карты (скруглённая карточка) (`f7a1842`).
- **Свёртка формы до peek** (только адреса + кнопки) (`4766bac`).
- **Hairline-подсвет** под формой убран (`272206a`).
- **Подвал / page-section:** исправлено отображение в обычном режиме (vh) (`dd11f74`).
- **Якорь формы** к краю viewport в обычном режиме (`d4093e0`).
- **Полировка переходов и состояний формы** (`9b74fcf`).
- **Мобильные профили 360–430px,** типографика и касания, баланс карта/форма на iPhone (`2cf3ac0`, `1609b4f`, `4a0db3d`, `f9c1ffd`).
- **Safe-area / старый Safari viewport** (`3eea7ae`).
- **Маркер центра Leaflet** без unpkg — локальный бандл (`a7ca61f`).
- **Аудит пассажира:** убран блокирующий `confirm()` перед сбросом пароля; скрыт захардкоженный «7 cars / ~5 min» до бэкенда; вычищен dev placeholder (`aba0aa4`, `f42f6a4`, `66f0f6d`).
- **Откат части UX:** обратно confirm и mock-подсказки рядом (`59b5db8`).
- **Крупные комментарии и `console.log`** убраны (`4681a61`, `bfe7c40`).

## OAuth / auth / config / cache

- **OAuth:** доверять pre-OAuth намерению над эхом бэкенда; роутер по ролям (`1d9be2f`). Детерминированный возврат Google OAuth с сохранением пассажирского флоу (`10fd842`). Принудительно запрошенный модуль даже при «не том» редиректе (`eab1dd9`).
- **Кэш / версия:** автообновление версии сборки без ручной очистки (`b49a1ae`).
- **Конфиг:** не сохранять `?config` в `localStorage` тихо (`b13280b`); при сбросе legacy — также вычищать черновик заказа (`ace506a`).
- **Bootstrap:** принять позднюю загрузку конфига после таймаута фолбэка; увеличены таймауты — upstream медленный но живой; UI не залипает при падении config-сервера (`ab0aa1a`, `96055e7`, `4ce3c29`).
- **Заказ после логина:** продолжить отложенный submit (`9af1382`).
- **Сброс заказа:** очистка from/to и storage при reset/logout (`b855131`).
- **Синхронизация** накопленных правок auth/register/modals/Vercel (`52ed3f0`).
- **Чат:** отключён polling из‑за протухшего TLS у upstream (`ae78fca`).
